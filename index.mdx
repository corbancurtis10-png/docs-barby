EZAnimation is a lightweight Roblox Luau library that gives you a clean, consistent API for every animation need — from simple one-shot plays with fade control, to crossfades, queues, ragdolls, and property tweens. Drop it into any Roblox project, point it at your animation ID list, and start animating in seconds.

## Quick Example

```lua EZAnimation basic usage
local EZAnim = require(game.ReplicatedStorage.EZAnim)

local Anim = EZAnim.init({
    List = {
        Idle   = "rbxassetid://1234567890",
        Run    = "rbxassetid://0987654321",
        Jump   = "rbxassetid://1122334455",
    }
})

-- Play the Idle animation with a 0.3s fade-in
Anim:play(character, "Idle", "Idle", true, 0.3)

-- Stop it with a 0.5s fade-out
Anim:stop(character, "Idle", 0.5)
```
